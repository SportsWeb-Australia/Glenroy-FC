# Glenroy FC — Website Concepts

Four design directions for the Glenroy Football Club (The Wallabies, EDFL) website, built from real club content and the official club crest.

- **/matchday** — bold, dark, match-centre led
- **/community** — bright, warm, welcoming
- **/heritage** — classic, premium, honour board
- **/editorial** — sports-media magazine

Static HTML. Landing page at `/index.html`. Built by Click Sports Media.
